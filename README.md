# 🦜 ARCHE3-Animal Desktop

**Official desktop application** for ARCHE3-Animal — a biologically-inspired AI parrot with a complete dopamine-based cognitive system.

---

## ✨ Features

- Real-time 3D African Grey Parrot visualization
- Full **Dopamine System v2** with 5 reward components:
  - Novelty
  - Weakness
  - Improvement
  - Ethical evaluation
  - Creation (4th sense)
- Dopamine-modulated gain visualization (0.5x – 10.0x)
- Step-by-step Reasoning Chain
- Live reward history chart with trend
- Dark / Light theme support
- Interactive tasks:
  - Explore
  - Mimic Sounds
  - Social Play
  - Rest
  - Free Roam

## 🚀 How to Run

1. Download and open the `.dmg` file  
2. Drag `ARCHE3-Animal.app` to **Applications**  
3. Launch the application  
4. Wait for model weights to download (~2 GB)  
5. Enjoy real-time interaction with the model
